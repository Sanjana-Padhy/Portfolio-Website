🌐 Personal Portfolio Website
This is a responsive, single-page personal portfolio website built using HTML, CSS, JavaScript, and jQuery, designed to showcase my projects, skills, education, and contact information.

🔧 Features
Fully responsive design for all screen sizes

Scroll-triggered animations using ScrollReveal.js

Sticky navigation bar with smooth scrolling

Downloadable resume (CV)

Live contact form connected to Google Sheets for submissions

Project section includes:

Simple Web Calculator

Cricket Game (Bat-Ball-Stump)

Myntra Clone Website

🛠️ Tech Stack
HTML5

CSS3

JavaScript / jQuery

ScrollReveal.js

Google Apps Script (for contact form submission)
